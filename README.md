# swagger

swaggerを試してみる

***
## Installation

install `multi-file-swagger`
> @see https://github.com/mohsen1/multi-file-swagger-example#usage

```sh
$ npm install -g multi-file-swagger
```

***
## Usage

build

- yaml

```sh
$ npm run build-yaml
```

- json

```sh
$ npm run build-json
```
